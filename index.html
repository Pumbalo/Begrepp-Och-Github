Github Flow

GitHub flow är ett grenbaserat arbetsflöde som GitHub har producerat. Detta arbetsflöde är inte bara användbart för utvecklare, utan vem som helst kan använda detta arbetsflöde. Med hjälp av GitHub flow så kan “teams” experimentera fritt och göra implementeringar regelbundet. Begreppen inom GitHub Flow är Branch, Commits, Pull Request, Review, Deploy och Merge.

Flexbox

Flexbox är ett CSS element som låter oss göra endimensionell layout modell. Detta element tillåter oss att ändra tex utrymme fördelningen mellan objekt. När man säger att Flexbox är endimensionell så betyder det att flexbox hanterar layouten i en dimension åt gången, rad eller kolumn. Det finns också en tvådimensionell metod, Grid. Grid styr kolumner och rader tillsammans.

DOM

Dom, som står för “Document Object Module”, är ett programmeringsgränssnitt för en hemsida. Med hjälp av DOM kan program ändra dokumentets struktur, stil och innehåll. DOM representerar dokumentet som noder och objekt så att programmeringsspråk kan interagera med sidan. Som en objektorienterad representation av webbsidan, kan den modifieras med ett skriptspråk som JavaScript.

Fin Kod

Fin Kod betyder att ens kod är lättläst och att den ska vara enkel att förstå av andra webbutvecklare/programmerare. Det finns en del tips för att skriva en fin kod. Detta är saker som tex lägga till kommentarer som kort förklarar funktionen, döpa sina variabler till namn som förklarar vad det är till för (tex class=”navigation” kan vara till navigation). Att förkorta sin kod så mycket som möjligt och att ej ha en massa onödig kod är också en viktig del i programmering.

Ramverk

Ett ramverk är en mjukvara som låter dig förenkla webbutveckling processen och göra det lättare att bygga en hemsida. Ramverk innehåller olika “mallar” som hjälper dig att programmera tex skript på en hemsida, fast på ett enklare och kortare sätt. Det kan vara allt från ha en “login session” till att animationer ska bete sig på ett specifikt sätt när något händer. Det finns många ramverk inom webbutveckling, men de som dom flesta vet om är React och Angular.

React är skapat av FaceBook och är ett JavaScript ramverk. Det används för att bygga så kallade “single page applications”, vilket betyder att hemsidan är i en och samma “page” även fast koden ändras på sidan. Hemsidan behöver aldrig laddas om för att nya funktioner ska ge effekt. React klarar av att göra både små och stora projekt.

Angular är skapat av Google och är baserat på TypeScript. Likadant som React så är Angular en “single page application”. För tillfället är Angular det näst mest populära front-end ramverket enligt Github och används av företag som PayPal, Google och Nike.
